# UnrealTPS

# 注意事項
- 名前の入力は、Shift+F1でマウスポインタを取り戻すことで可能になります。

# 自分で実装した要素（教科書を見つつの実装）
- UI
  - タイトル画面
  - メインゲーム内での制限時間と得点の表示
  - リザルト画面
  - ランキング画面
- 行動
  - 攻撃（Spaceで球が出ます）
- システム
  - 得点（敵を倒すと追加されます。一体ごとのレートは100です）
  - 制限時間（デフォルトでは10秒、エディタ内で変更できるようになっています）
  - ランキング（1~10位まで。ランクインすると登録できます）
  - セーブ・ロード（自動。エディタでの起動も対象です）
